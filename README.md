## 项目描述
### 技术栈
Vue2.0全家桶 + axios +　Vuex +  Mint-Ui 

## 主要依赖：
1. Vue2.0 前端页面展示
2. Axios 异步数据的请求
3. flexible.js 解决移动端设备兼容
4. Mint-ui 移动端组件库
5. SPA单页应用，前端后分离
6. localStorge实现本地数据存储

## 实现功能：
1. tabbar切换
2. 搜索框，根据输入关键字查询数据
3. 二维数组菜单选择
4. 数据传递
5. tabContainer的切换 
6. 根据不同选择数据进行数据查询，更新
